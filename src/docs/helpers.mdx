# Helpers

import { Stack } from '@mantine/core'
import { HelperCard } from './helper-card'

The helpers is a way that handlebars provides for convinent. There are three types:

- Handlebars builtin helpers
- Rxofclock builtin helpers
- User defined helpers

## Handlebars builtin helpers

Please visit [the official doc](https://handlebarsjs.com/guide/builtin-helpers.html) for more info

## Rxofclock builtin helpers

### String helpers

<Stack>
  <HelperCard
    title="trim"
    description="Remove white spaces before and end in a string"
    usage="{{trim [colName]}}"
    value="'  some value  '"
    output="'some value'"
  />
  <HelperCard
    title="trimStart"
    description="Remove white spaces before a string"
    usage="{{trimStart [colName]}}"
    value="'  some value  '"
    output="'some value  '"
  />
  <HelperCard
    title="trimEnd"
    description="Remove white spaces end in a string"
    usage="{{trimEnd [colName]}}"
    value="'  some value  '"
    output="'  some value'"
  />
</Stack>

### Number helpers

<Stack>
  <HelperCard
    title="toFixed(n)"
    description="Convert a number to fixed string"
    usage="{{toFixed [colName] 2}}"
    value="31.25367"
    output="31.25"
  />
</Stack>

### Random helpers
