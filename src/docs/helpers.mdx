# Template

import { Stack } from '@mantine/core'
import { HelperCard } from './helper-card'
import { DocLink } from './doc-link'

Use the template editor to shape the text to generate. The basic usage of the column value is `{{[columnName]}}`. For example, you can use the template below to generate the shape of 'hello xxx(the column values)':

```handlebars
hello {{[userName]}}
```

You can use the `@data` variables in your template(<DocLink href="https://handlebarsjs.com/api-reference/data-variables.html#root">docs here</DocLink>):

- `@root`: Initial context with which the template was executed. `{{#each array}} {{@root.foo}} {{/each}}`
- `@first`: Set to true by the each helper for the first step of iteration. `{{#each array}} {{#if @first}} First! {{/if}} {{/each}}`
- `@index`: Zero-based index for the current iteration step. Set by the `each` helper. `{{#each array}} {{@index}} {{/each}}`
- `@key`: Key name for the current iteration step. Set by the each helper when iterating over objects. `{{#each array}} {{@key}} {{/each}}`
- `@last`: Set to true by the each helper for the last step of iteration. `{{#each array}} {{#if @last}} Last :( {{/if}} {{/each}}`

## Helpers

The helpers is a way that handlebars provides for convinent. There are three types:

- Handlebars builtin helpers
- Rxofclock builtin helpers
- User defined helpers

### Handlebars builtin helpers

Please visit <DocLink href="https://handlebarsjs.com/guide/builtin-helpers.html">the official doc</DocLink> for more info

### Rxofclock builtin helpers

#### String helpers

<Stack>
  <HelperCard
    title="trim"
    description="Remove white spaces before and end in a string"
    usage="{{trim [colName]}}"
    value="'  some value  '"
    output="'some value'"
  />
  <HelperCard
    title="trimStart"
    description="Remove white spaces before a string"
    usage="{{trimStart [colName]}}"
    value="'  some value  '"
    output="'some value  '"
  />
  <HelperCard
    title="trimEnd"
    description="Remove white spaces end in a string"
    usage="{{trimEnd [colName]}}"
    value="'  some value  '"
    output="'  some value'"
  />
</Stack>

#### Number helpers

<Stack>
  <HelperCard
